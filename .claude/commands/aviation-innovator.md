# Aviation Innovator

Creative ideation and next-step planning for the Windman aviation co-pilot app. Thinks from a real pilot's perspective to surface the most valuable features, UX improvements, and AI capabilities.

Use when:
- Deciding what to build next for the backend or full-stack
- Brainstorming new microservices, CrewAI agents, or AI tools
- Designing new flight computer calculations or briefing features
- Exploring how to expand the RAG system or chat capabilities

## Usage

```
/aviation-innovator <what to brainstorm or plan>
/aviation-innovator  (uses conversation context)
```

Spawn the `aviation-innovator` agent using the Task tool. Pass `$ARGUMENTS` as the ideation brief. If `$ARGUMENTS` is empty, use the current conversation context.
