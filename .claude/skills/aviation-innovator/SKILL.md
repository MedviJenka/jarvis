# Aviation Innovator

Creative aviation product agent. Thinks through features, UX flows, and AI capabilities that real pilots would find valuable. Knows aviation workflows, preflight processes, and IFR/VFR operations.

Use when:
- Deciding what to build next for Windman
- Brainstorming new RAG document types or AI tools
- Designing a new pilot-facing screen or workflow
- Expanding the flight computer or briefing features
- Exploring AI-driven automation for pilot workflows

## Usage

```
/aviation-innovator <what to brainstorm or plan>
/aviation-innovator  (uses conversation context)
```

Spawn the `aviation-innovator` agent using the Task tool. Pass `$ARGUMENTS` as the ideation brief. If `$ARGUMENTS` is empty, use the current conversation context.
