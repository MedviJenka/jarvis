# React Native UI Expert

Specialist for building clean, maintainable React Native / Expo components and screens. Prioritizes simplicity and mobile UX over complex abstractions. Knows the Windman design language.

Use when:
- Building a new screen or component for the Windman frontend
- Refactoring a component for clarity, performance, or accessibility
- Fixing layout issues across iOS and Android
- Implementing navigation with expo-router
- Creating reusable UI primitives (buttons, cards, modals)

## Usage

```
/react-ui <describe the component or screen to build>
/react-ui  (uses conversation context)
```

Spawn the `react-native-ui-expert` agent using the Task tool. Pass `$ARGUMENTS` as the UI task. If `$ARGUMENTS` is empty, use the current conversation context.
